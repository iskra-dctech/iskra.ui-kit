# @iskra-ui/tokens

DTCG design tokens for **Искра.DCI** — compiled to CSS variables, JSON and a typed TypeScript map via Style Dictionary.

## Install

```bash
pnpm add @iskra-ui/tokens
```

Usually consumed through `@iskra-ui/styles` rather than directly.

## Usage

```ts
import tokens from '@iskra-ui/tokens/tokens.json'
```

Entry points: `@iskra-ui/tokens`, `@iskra-ui/tokens/tokens.css`, `@iskra-ui/tokens/tokens.json`, `@iskra-ui/tokens/src/*`.
