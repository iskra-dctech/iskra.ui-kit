# @iskra-ui/styles

Global stylesheet for **Искра.DCI** — design tokens, self-hosted fonts (Inter, JetBrains Mono), reset and element classes.

## Install

```bash
pnpm add @iskra-ui/styles
```

## Usage

Import once at your app entry, before component styles:

```ts
import '@iskra-ui/styles/index.css'
```

Granular entry points: `tokens.css`, `fonts.css`, `reset.css`, `elements.css`.
