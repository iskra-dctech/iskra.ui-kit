# @iskra-ui/core

Headless core for **Искра.DCI** — framework-agnostic state machines, ARIA helpers, keyboard navigation and focus trap utilities shared by React and Vue packages.

## Install

```bash
pnpm add @iskra-ui/core
```

Usually imported indirectly through `@iskra-ui/react` or `@iskra-ui/vue`. Use directly when building custom components on the same logic.

## Usage

```ts
import { createId, createTabsIds, getNextTabValue } from '@iskra-ui/core'
```
